# Pilot Weather Briefing Agent

## Summary
Build a custom pilot weather briefing agent that gathers preferred aviation weather information, organizes it into a personalized briefing, and reads it back aloud. This could also become an iPhone app.

## Problem
Standard weather briefings can be noisy, generic, and not tailored to a pilot's preferred sources, flight profile, or decision-making style. Visual tools such as ForeFlight are strong for scanning weather graphics, but sometimes the pilot may want an audio briefing first and then use visuals to investigate areas of concern.

## Proposed Automation
Create an agent that collects aviation weather data from preferred sources, assembles a custom weather briefing, highlights relevant risks and conditions, and reads the briefing back in audio form. The workflow should support an audio-first experience, followed by visual review when the briefing identifies areas that need closer attention. The system could begin as an agent workflow and later expand into an iPhone app.

The source mix should include the best FAA resources plus trusted instructor recommendations gathered from podcasts and YouTube.
